# 네이버 커머스 API 센터

> 네이버 커머스 플랫폼의 판매자·솔루션 운영에 필요한 공식 API 문서를 한곳에서 제공합니다. 인증·상품·주문·정산·문의·물류·커머스솔루션·판매자정보·API데이터솔루션 등 핵심 도메인의 endpoint 별 자연어 본문과 구조 데이터(요청·응답·에러·예시) 를 LLM 이 바로 소비하기 적합한 마크다운으로 노출합니다. AI 도구·에이전트는 본 인덱스에서 도메인을 좁힌 뒤 endpoint 단위 .md 문서를 따라가 상세 스펙을 탐색할 수 있습니다.



## 공통소개
- [AI활용가이드](https://apicenter.commerce.naver.com/llms/intro-AI활용가이드.md): AI 도구·LLM 에서 커머스API 문서를 효율적으로 탐색하는 방법과 llms.txt 활용 방안을 안내합니다.
- [RESTful API](https://apicenter.commerce.naver.com/llms/intro-restful-api.md): RESTful API 규격(Content-Type, 인코딩, 날짜 형식, HTTP 상태 코드, 오류 응답 형식) 을 정의합니다.
- [문제 해결](https://apicenter.commerce.naver.com/llms/intro-문제해결.md): 인증 오류·권한 오류·요청량 초과 등 자주 발생하는 문제와 해결 방법을 안내합니다.
- [인증](https://apicenter.commerce.naver.com/llms/intro-인증.md): 커머스API 인증 절차(애플리케이션 등록·OAuth 2.0 토큰 발급·전자서명 생성) 를 설명합니다.
- [제약 사항](https://apicenter.commerce.naver.com/llms/intro-제약사항.md): TLS 버전·권한 그룹·요청량 제한(rate limit, quota limit) 등 API 호출 시 적용되는 제약 사항을 정의합니다.
- [커머스API 소개](https://apicenter.commerce.naver.com/llms/intro-커머스API소개.md): 커머스API 의 정의와 사용 흐름(가입·애플리케이션 등록·게이트웨이 호출·인증) 을 소개합니다.

- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/all/detail - 전체 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-all-detail.md): 채널의 마케팅 성과를 채널·매체 합산 형태로 조회합니다. 광고 채널 비교/회의 자료 자동화에 적합하며 후행 보정으로 최근 결과가 변동될 수 있습니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/custom/detail - 사용자 정의 채널 상세 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-custom-detail.md): 사용자 정의 마케팅 채널의 상세 성과를 조회합니다. 자체 분류 채널 그룹의 지표 노출에 사용하며 사용자 정의 채널 코드 매핑이 사전에 필요합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/custom/simple - 사용자 정의 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-custom-simple.md): 사용자 정의 마케팅 채널의 핵심 지표만 간단히 반환합니다. 대시보드 위젯·일일 알림처럼 빠른 응답이 필요한 시나리오에 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/hourly/detail - 시간대별 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-hourly-detail.md): 마케팅 성과를 시간대별 상세 지표로 조회합니다. 프로모션 모니터링·광고 비드 자동화 데이터로 활용하며 직전 1~2시간은 보정 가능성이 있습니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/hourly/simple - 시간대별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-hourly-simple.md): 마케팅 성과의 시간대별 핵심 지표를 간단히 반환합니다. 응답 부하가 작아 빠른 모니터링 대시보드와 알림 트리거에 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/search/detail - 검색 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-search-detail.md): 검색 채널 마케팅 성과 상세 지표를 조회합니다. 검색광고 운영 회의 자료 자동화·검색 채널 BI 적재에 활용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/search/keyword - 검색 채널 키워드별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-search-keyword.md): 검색 채널 유입의 키워드별 성과를 조회합니다. 키워드 매출 기여 분석·검색광고 입찰 자동화의 입력 데이터로 활용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/website/daily - 웹사이트 채널 일별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-website-daily.md): 외부 웹사이트 채널 유입의 일별 마케팅 성과를 조회합니다. 배너/제휴 트래픽 일별 추이 적재·일일 리포트 자동화에 사용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/marketing/website/detail - 웹사이트 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-marketing-website-detail.md): 웹사이트 채널 유입의 채널·매체 합산 마케팅 성과를 조회합니다. 매체 비교/매출 기여 분석에 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/realtime/daily - 오늘 보고서 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-realtime-daily.md): 오늘자 실시간 채널 보고서를 조회합니다. 실시간 매출/방문 모니터링과 알림 트리거에 적합하며 폴링 주기는 1~5분이 권장됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/delivery/detail - 배송 통계 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-delivery-detail.md): 배송 단계별 판매 통계를 조회합니다. 발송 지연/반품률 분석·배송 운영 KPI 대시보드에 활용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/hourly/detail - 판매 성과 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-hourly-detail.md): 시간대별 판매 성과 상세 지표를 조회합니다. 프로모션 효과 분석·라이브 매출 모니터링에 적합하며 직전 시점은 보정 가능성이 있습니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/product/detail - 상품 성과 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-product-detail.md): 상품 단위 판매 성과 상세 지표를 조회합니다. 상품별 매출 알림·베스트셀러 큐레이션 자동화의 입력 데이터로 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/product-marketing/category - 상품/마케팅 채널 카테고리(소)별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-product-marketing-category.md): 상품×마케팅 채널 매출을 카테고리(소) 단위로 합산해 조회합니다. 카테고리별 마케팅 ROI 분석·카테고리 비중 리포트에 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/product-marketing/detail - 상품/마케팅 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-product-marketing-detail.md): 상품×마케팅 채널 단위 매출 상세 지표를 조회합니다. 상품별 광고 채널 매출 기여 분석·자동 보고서 생성에 사용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/product-search/detail - 상품/검색 채널 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-product-search-detail.md): 상품×검색 채널 단위 매출 상세 지표를 조회합니다. 검색 유입 상품 매출 분석·검색광고 회의 자료 자동화에 적합합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/sales/product-search/keyword-by-product - 상품/검색 채널 상품별 키워드 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-sales-product-search-keyword-by-product.md): 상품별 검색 채널 유입 키워드 매출을 조회합니다. 검색광고 입찰/광고문 최적화 데이터로 활용되며 결과가 매우 클 수 있어 분할 호출이 필요합니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/shopping/page/detail - 페이지별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-shopping-page-detail.md): 페이지별 쇼핑행동 지표를 조회합니다. 카테고리/기획전 페이지 유입·전환 분석과 페이지 개선 효과 측정에 사용됩니다.
- [GET /v1/bizdata-stats/channels/{channelNo}/shopping/product/detail - 상품별 API](https://apicenter.commerce.naver.com/llms/get-v1-bizdata-stats-channels-channelNo-shopping-product-detail.md): 상품별 쇼핑행동 지표를 조회합니다. 노출/조회/장바구니/구매 전환 흐름 분석과 상위 상품 큐레이션에 활용됩니다.
- [GET /v1/customer-data/customer-status/account/statistics - 고객 현황(계정) API](https://apicenter.commerce.naver.com/llms/get-v1-customer-data-customer-status-account-statistics.md): 계정 단위(전 채널 합산) 고객 현황 통계를 조회합니다. 다중 채널 통합 추이 분석·임원 보고용 KPI 대시보드 자동화에 적합합니다.
- [GET /v1/customer-data/customer-status/channels/{channelNo}/statistics - 고객 현황(채널) API](https://apicenter.commerce.naver.com/llms/get-v1-customer-data-customer-status-channels-channelNo-statistics.md): 특정 채널의 고객 현황 통계를 조회합니다. 채널 단위 활성/이탈 고객·신규 유입 추이 BI 적재에 활용되며 결과는 후행 보정될 수 있습니다.

- [POST /v1/pay-merchant/inquiries/{inquiryNo}/answer - 고객 문의 답변 등록](https://apicenter.commerce.naver.com/llms/post-v1-pay-merchant-inquiries-inquiryNo-answer.md): 네이버페이 결제 문의에 신규 답변을 등록합니다. 챗봇·답변 자동화 도구가 inquiryNo 별 답변을 일괄 등록하는 시나리오에 적합합니다.
- [PUT /v1/contents/qnas/{questionId} - 상품 문의 답변 등록/수정](https://apicenter.commerce.naver.com/llms/put-v1-contents-qnas-questionId.md): 상품 문의 답변을 등록 또는 수정합니다. 자동화 도구가 템플릿 답변을 적용하거나 일괄 답변 처리하는 시나리오에 사용됩니다.
